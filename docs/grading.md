# Grading

Advancement is competency-based. Learners move forward by passing explicit gates with artifacts.

## Weights

| Component | Weight |
| --- | ---: |
| Weekly labs | 40% |
| Build assignments | 35% |
| Checkpoints and reflections | 15% |
| Professional delivery quality | 10% |

## Pass Criteria

- Minimum 80/100 on weighted rubric.
- No zero in critical criteria (functionality, reliability, documentation).
- Required artifacts submitted and reproducible.
