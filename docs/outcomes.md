# Outcomes

By program completion, learners are expected to ship portfolio-ready work and demonstrate job-ready competencies.

## Program Outcomes

1. Ship 6-8 AI-powered tools for SMB workflows.
2. Demonstrate API integration, structured LLM output, and deployment literacy.
3. Produce portfolio artifacts: case studies, demos, and scoped offers.
4. Pass competency gates tied to measurable technical and business criteria.

## Evidence of Completion

- Working repositories and deployment links
- Rubric-scored submissions
- Portfolio package in `portfolio/`
