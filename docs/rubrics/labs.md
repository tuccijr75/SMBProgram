# Labs Rubric

## Weighting

| Criterion | Weight |
| --- | ---: |
| Technical correctness | 35% |
| Completion of required steps | 25% |
| Documentation quality | 20% |
| Verification evidence | 20% |

## Scoring Guidance

- 4: Exceeds expected quality with strong evidence
- 3: Meets requirements with minor gaps
- 2: Partially complete; material issues remain
- 1: Incomplete or not reproducible

## Pass Threshold

- Weighted score >= 80/100
- No score below 2 in technical correctness or verification evidence
